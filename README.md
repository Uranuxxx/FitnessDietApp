# FitnessDietApp

## 概要
FitnessDietAppは、身長、体重、目標カロリーを基に、食事プランを提案するAndroidアプリです。

## 使用方法
1. アプリを開きます。
2. 必要な情報（体重、身長、目標カロリー）を入力します。
3. 「計算」ボタンを押すと、推奨される食事プランが表示されます。

## 開発環境
- **言語**：Java
- **フレームワーク**：Android SDK
- **IDE**：Android Studio
- **最小SDK**：21 (Android 5.0 Lollipop)

## 機能
- 食事プランの提案
- 簡単な栄養計算
